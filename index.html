התקנת BUILD YOUR BRAND

יוצרים נוכחות דיגיטלית עם ערכת מותג מלאה

הפלאגין BUILD YOUR BRAND מייצר עבורכם ערכת מותג אישי מלאה. בדף זה תוכלו להתקין או להוריד אותו, והוראות השימוש המלאות נמצאות בשיעור.

התקנה ישירה

ההתקנה מתבצעת ישירות מממשק המשתמש, לא באמצעות הרצת פקודה. ב-Cowork:

1
התאמה אישית ← פלאגינים
2
פלאגינים אישיים ← + ← הוספת ספריית-פלאגינים ← benefits-il/brand-studio ← סיור בפלאגינים
3
התקינו את BUILD YOUR BRAND

שימו לב: ספריית-פלאגינים תופיע רק לאחר התקנת פלאגין אחד לפחות. התקינו פלאגין כלשהו תחילה, או הוסיפו ישירות את התיקייה שהורדתם.

הורדה ידנית

מעדיפים לא להוסיף ספריית-פלאגינים? הורידו את הפלאגין הארוז והוסיפו את התיקייה ישירות תחת פלאגינים אישיים.

הורדת הפלאגין

לצורך יצירת תמונות הקרוסלה יש לחבר את Higgsfield, בחינם ובאופן חד-פעמי. מדריך השימוש המלא נמצא בשיעור.

כיצד לעבוד עם הפלאגין ← בשיעור הקורס.

Install BUILD YOUR BRAND

Build your online presence with a complete brand pack

The BUILD YOUR BRAND plugin builds a complete personal brand pack for you. Install or download it from this page, and find the full instructions in the course lesson.

Direct installation

The plugin is installed directly from the user interface, not by running a command. In Cowork:

1
Customize → Plugins
2
Personal plugins → + → Add marketplace → type benefits-il/brand-studio → Browse plugins
3
Install BUILD YOUR BRAND

Please note: the plugins library only appears after at least one plugin is already installed. Install any plugin first, or add the downloaded folder directly.

Manual download

Prefer not to add a library? Download the packaged plugin and add the folder directly under Personal plugins.

Download Plugin

To use it, you will also need to connect Higgsfield once for free to generate the carousel images. The complete guide is in the lesson.
