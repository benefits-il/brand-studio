Plugin · generic engine

One plugin. Any brand.

brand-studio is one plugin that does five jobs: an outreach email, a welcome email, a post, a carousel, and a video. Hand it a brand pack and it turns raw drafts into finished work in that brand's look. It carries no brand of its own. Swap the pack, and the same plugin works for another brand.

The four parts

A plugin is the box. It wraps three familiar parts and opens one door to the outside. That is the whole idea.

Skills

Five — one per job

The method. Each skill reads the brand sheet you give it — colors, type, mark, voice — and applies it. Fully generic: swap the brand pack and it works for another brand.

Connectors

Two, wired two ways

Higgsfield for image and video, wired in .mcp.json. Gmail for drafts, turned on once in Connectors. The door to the outside.

Commands

Five

The buttons. /outreach /welcome /post /carousel /reel — each one drives the right agent.

Sub-agents

Two

The personas. content-writer writes the emails and captions; visual-producer makes the images, the carousel, and the video.

Plugin is the box. It wraps the other three. Install it once and you get them all together.

Install in Cowork

In Cowork, open /plugin and run the two lines:

/plugin marketplace add benefits-il/brand-studio /plugin install brand-studio@brand-studio

Add each connector once in your Connectors list (claude.ai: Settings then Connectors · Cowork: Customize then Connectors). Higgsfield (image and video): choose Add custom connector, name it Higgsfield, paste the URL https://mcp.higgsfield.ai/mcp, click Connect, and sign in with your Higgsfield account — free, no card, no API key. The plugin already points to this URL in .mcp.json; this sign-in is what authorizes it. Gmail (drafts): turn on the built-in connector and sign in. Emails are saved as drafts, never sent.

How the brand gets in

The plugin reads only what you attach. No brand is baked into the code. Attach three things and run a command:

1
Voice guidelines (brand-voice.md) — how the brand sounds. Generate it from your own materials with a voice plugin, or write it by hand.
2
Brand sheet (brand-sheet.md) — the colors, the mark, the facts.
3
A brief for the job you picked — what the piece needs to say.

The same plugin works for any brand whose pack you hand it. A full demo pack to practice on lives on a separate page.
