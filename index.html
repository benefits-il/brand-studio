מתקינים דרך הממשק — לא מקלידים פקודת-התקנה. ב-Cowork:

אם Personal plugins נראה ריק ואין כפתור Add marketplace: ה-marketplace מופיע רק אחרי שכבר מותקן לפחות plugin אחד. מתקינים קודם plugin כלשהו (או מוסיפים את ה-.zip מעמוד-ההורדה), ואז האפשרות Add marketplace נפתחת. זו תקלת-פעם-ראשונה ידועה, לא בעיה ב-plugin.

שני החיבורים — מוסיפים כל אחד פעם אחת ברשימת ה-Connectors (ב-claude.ai: Settings ואז Connectors · ב-Cowork: Customize ואז Connectors). Higgsfield (תמונה ווידאו): בוחרים Add custom connector, קוראים לו Higgsfield, מדביקים את הכתובת https://mcp.higgsfield.ai/mcp, לוחצים Connect, ומתחברים עם חשבון Higgsfield — חינם, בלי כרטיס, בלי API key. ה-plugin כבר מפנה לכתובת הזו ב-.mcp.json; ההתחברות היא מה שמרשה לו לעבוד. Gmail (טיוטות): מדליקים את החיבור המובנה ומתחברים. המיילים נשמרים כטיוטה, אף פעם לא נשלחים. כדי להתחיל: מריצים /start-here.

Plugin · BUILD YOUR BRAND

Three questions. A whole brand.

Install the plugin, run start-here, and answer three short questions about yourself. It builds a full brand pack — voice, design system, email, carousel, and a landing page — into a folder you choose, with nothing to download. All about your own world. The plugin carries no brand of its own; it builds yours.

The four parts

A plugin is the box. It wraps three familiar parts and opens one door to the outside. That is the whole idea.

Skills

One per job

The method. start-here orchestrates them all — brand voice, design system, email, carousel, and landing page. Each skill reads the brand sheet it built and applies it, so everything comes out consistent.

Connectors

Two, wired two ways

Higgsfield for image and video, wired in .mcp.json. Gmail for drafts, turned on once in Connectors. The door to the outside.

Commands

The buttons

/start-here is the front door. /outreach /welcome /post /carousel /reel stay available for each job on its own.

Sub-agents

Two

The personas. content-writer writes the emails and captions; visual-producer makes the images, the carousel, and the video.

Plugin is the box. It wraps the other three. Install it once and you get them all together.

Install in Cowork

Install through the interface — you don't type an install command. In Cowork:

1
Go to Customize, then the Plugins tab.
2
Under Personal plugins click the +, choose Add marketplace, type benefits-il/brand-studio, and click Browse plugins.
3
Install BUILD YOUR BRAND.

If "Personal plugins" looks empty and there's no "Add marketplace": a marketplace only shows up once at least one plugin is already installed. Install any plugin first (or add the .zip from the download page), and the Add marketplace option appears. Known first-time snag, not a plugin problem.

Add each connector once in your Connectors list (claude.ai: Settings then Connectors · Cowork: Customize then Connectors). Higgsfield (image and video): choose Add custom connector, name it Higgsfield, paste the URL https://mcp.higgsfield.ai/mcp, click Connect, and sign in with your Higgsfield account — free, no card, no API key. The plugin already points to this URL in .mcp.json; this sign-in is what authorizes it. Gmail (drafts): turn on the built-in connector and sign in. Emails are saved as drafts, never sent. To begin, run /start-here.

What start-here builds for you

You answer three questions, and the plugin builds the whole pack into a folder you choose — nothing to attach or download:

1
Brand voice (brand-voice.md) — how your brand sounds, derived from your answers.
2
Design system (design-system.md) — auto-picked from six polished presets, so it always comes out clean.
3
Email, carousel, and a landing page — finished, in your voice and your design.

All about your own world. Edit any file, or re-run any single step.
